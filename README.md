# HealthConnect App

## 📌 Overview

HealthConnect is a full-stack healthcare and fitness platform that connects patients, doctors, and fitness experts. The system provides personalized health recommendations, appointment booking, and community interaction.

---

## 🚀 Features

* Multi-role authentication (Patient, Doctor, Fitness Expert)
* Personalized health recommendations
* Exercise and diet suggestions
* Appointment booking system
* Doctor-to-doctor recommendations
* Google Maps integration for nearby doctors
* Subscription-based premium content
* Public health forum

---

## 🛠️ Tech Stack

* Frontend: React
* Backend: Node.js + Express
* Database: MongoDB
* APIs: Google Maps API

---

## 📂 Project Structure

* `/backend` → REST API & business logic
* `/frontend` → User interface
* `/docs` → Documentation
